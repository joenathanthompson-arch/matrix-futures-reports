# Matrix Trading Reports

Daily market analysis reports generated by Manus AI for the Matrix Trading System.

## Structure

```
matrix-reports/
├── reports/
│   └── YYYY/
│       └── MM/
│           └── YYYY-MM-DD.md       # Daily analysis report
├── data/
│   └── bias_scores/
│       └── YYYY-MM-DD.json         # Machine-readable bias scores
└── README.md
```

## Report Format

Each daily report includes:
- Executive Summary
- Market Bias Scorecard (per instrument)
- Detailed Instrument Analysis
- Economic Calendar
- Latest Market News
- References

## Integration

- **Manus AI**: Commits daily reports and bias scores
- **Portfolio Manager**: Reads `data/bias_scores/YYYY-MM-DD.json` for budget allocation
- **Trader EAs**: Receive strategy recommendations via Telegram from PM

## Signal Thresholds

| Score Range | Signal | Recommended Action |
|-------------|--------|-------------------|
| >= 15 | Strong Bullish | IB_BREAKOUT long bias |
| 5 to 14 | Bullish | IB_BREAKOUT cautious |
| -4 to 4 | Neutral | IB_REVERSION or NO_TRADE |
| -14 to -5 | Bearish | IB_BREAKOUT short bias |
| <= -15 | Strong Bearish | IB_BREAKOUT short bias |

## Usage by Portfolio Manager

The PM reads the latest bias scores JSON and:
1. Adjusts margin budgets per symbol
2. Sends strategy recommendations to Trader chats
3. Can override with `/pm budget GC 25` commands

## Historical Data

Reports are retained for:
- Backtesting strategy performance vs predictions
- Training future AI models
- Audit trail of trading decisions
